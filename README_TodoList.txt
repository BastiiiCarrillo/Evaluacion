
Mi lista de tareas – Aplicación Web con Vue.js

Descripción del problema
Muchas personas tienen dificultades para organizar sus tareas diarias, lo que puede llevar a olvidos, desorganización y baja productividad. Es necesario contar con una herramienta simple y accesible que permita anotar y gestionar tareas pendientes de forma clara y práctica.

Solución propuesta
Esta aplicación es una lista de tareas (ToDo List) desarrollada como una SPA (Single Page Application) utilizando Vue.js. Permite agregar nuevas tareas, marcarlas como completadas o eliminarlas. 
Además, guarda la información en el navegador mediante `localStorage`, lo que permite mantener los datos incluso al cerrar la ventana. Como elemento adicional de valor, al iniciar la aplicación se muestra una frase motivacional obtenida desde una API pública para incentivar al usuario.

Tecnologías utilizadas
- Vue 3 (Composition API)
- Vite
- JavaScript
- HTML5 / CSS3
- LocalStorage
- API externa: https://api.quotable.io/random


Instrucciones para ejecutar la aplicación
1. Clona este repositorio:
   git clone https://github.com/tuusuario/todo-list-vue.git

2. Entra a la carpeta del proyecto:
   cd todo-list-vue

3. Instala las dependencias:
   npm install

4. Ejecuta la aplicación localmente:
   npm run dev

5. Abre tu navegador en:
   http://localhost:5173
